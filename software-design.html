Software Design and Engineering

Artifact: Travlr Full-Stack Web Application

The Travlr application is a full-stack web application developed using Node.js, Express, MongoDB, and Handlebars. The system allows users to view travel packages and manage trip information through a structured web interface. This project demonstrates my ability to design and implement a complete software system using the Model-View-Controller (MVC) architecture.

Enhancements Made

During the enhancement phase, I improved the functionality and organization of the application by refining routing logic, improving controller structure, and ensuring that data is properly retrieved and displayed from the database. I also enhanced the user interface to make navigation clearer and improve the overall user experience. These updates strengthened the maintainability, scalability, and usability of the application.

Skills Demonstrated

  • Software architecture using MVC design pattern
  • Full-stack web development
  • Routing and controller implementation
  • System functionality testing and debugging
  • User interface design and application structure
