Algorithms and Data Structures

Artifact: Contact Service Application

The Contact Service Application was developed in Java to manage contact records such as contact IDs, first names, last names, phone numbers, and addresses. This project demonstrates my ability to use data structures and algorithms to store, retrieve, and update information efficiently within a software system.

Enhancements Made

For the enhancement phase, I improved the Contact Service application by adding a search method that uses a linear search algorithm to locate a contact by ID within the ArrayList data structure. This enhancement improved the functionality of the application and demonstrated my understanding of how algorithms are applied to real programs to retrieve and manage data.

Skills Demonstrated

  • Use of ArrayList data structure
  • Linear search algorithm implementation
  • Data retrieval and update logic
  • Object-oriented programming in Java
  • Validation and data management
