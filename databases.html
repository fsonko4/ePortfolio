Artifact: Travlr Web Application Database Enhancement

The Travlr application uses MongoDB and Mongoose to store and retrieve travel destination data. This project demonstrates my ability to work with databases in a full-stack web application by connecting backend logic to structured data stored in a database.

Enhancements Made

For the enhancement phase, I improved the database functionality of the Travlr application by adding search and filtering capabilities to the trips retrieval process. I updated the controller logic so users can search for trips by name and filter results by maximum price using query parameters. This enhancement improved how data is retrieved from the database and made the application more useful and user-friendly.

Skills Demonstrated

  • MongoDB database integration
  • Mongoose schema design
  • Backend data retrieval using queries
  • Search and filtering with query parameters
  • Full-stack application development
